# Impeccable Design Context

## Design Context

### Users
Tabletop RPG hobbyists — D&D players, dungeon masters, and game designers — creating custom card sets at home for printing. Used at a desk, typically in the evening. The task is creative and deliberate: composing a card takes thought, so the tool should feel like a worthy instrument, not a disposable utility.

### Brand Personality
**Arcane · Crafted · Deliberate**

The interface should feel like a master craftsman's workbench — purposeful, unhurried, with a sense of weight and materiality. Gold and deep shadow are the core aesthetic signals. Not whimsical, not corporate. The tool makes physical objects (printed cards), so the UI should feel tactile.

### Aesthetic Direction
- **Theme:** Dark by default (users work at night; dark surfaces let the card previews breathe)
- **Palette:** Deep purples and near-blacks (#0e0c14, #1e1a2a) with gold accent (#c9a84c / #e4c878)
- **Typography:** Alegreya SC (already loaded) for labels and brand — ties the interface to the card aesthetic. System-ui for controls and body text.
- **Separators:** Diamond ornament glyphs (◆) in gold, not generic divider lines
- **Brand mark:** Nested diamond SVG in gold — a simplified "card gem"
- **Top border:** 1px gold stripe on the topbar header — a subtle crown

### Design Principles
1. **The interface serves the card** — UI chrome should recede; the card preview is the hero
2. **Gold is earned** — use it sparingly: accents, labels, the primary CTA. Never for decoration alone
3. **Labels teach the vocabulary** — section titles in Alegreya SC tie toolbar language to the card world
4. **No chrome for chrome's sake** — every element must earn its pixel real estate
5. **Tactile states** — hover, active, and focus states should feel physical, not digital-flat
